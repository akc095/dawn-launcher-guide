# Dawn Launcher 怎么用？桌面图标太乱怎么整理？免费快捷启动工具上手教程（附 Dawn Launcher 1 和 2 区别说明）

桌面右下角的图标已经堆到第三屏了，你点开始菜单找一个软件，得在几十个文件夹里翻半天。这种情况大概每个用 Windows 久了的人都遇到过。

我换过几款桌面整理工具，最后留下来的是 **Dawn Launcher**。这是一款国产的 Windows 快捷启动工具，免费、开源，作者是受不了自己桌面乱才动手写的。今天就说说它实际怎么用，能解决什么问题，以及网上常说的"Dawn Launcher 2"到底是什么。

## Dawn Launcher 是什么

简单定义一下：Dawn Launcher 是一个 Windows 桌面快捷启动工具，把你常用的软件、文件夹、网址收进一个分类面板里，按快捷键弹出来，点一下就打开，用完自动收起。

它的思路参考了早些年那款经典的"音速启动"。功能不复杂，就是帮你把散落一桌面的图标，变成一个可以分组、可以搜索的清单。

软件用 Electron 写的，这也是它体积偏大的原因——作者自己在说明里也提到了这点，并不是优点，是个取舍。如果你对软件体积特别敏感，这一点要心里有数。

## Dawn Launcher 能解决什么问题

先说说它实际能干啥，免得你下载完发现不是想要的东西。

- **整理散乱图标**：把程序快捷方式、磁盘分区、文件夹、网址统一收到分类面板里
- **快速搜索定位**：输入几个字母或汉字就能定位到目标软件，不用一个个翻
- **关联文件夹**：把常用的项目文件夹关联进来，文件夹里的内容会同步显示，不用手动维护
- **自定义外观**：主题、背景图、图标大小、字体都能调
- **进阶操作**：可以设置环境变量、调用命令行、添加微软商店应用——这条主要是给开发者和重度用户用的

办公场景下，把 Word、Excel、钉钉这些天天开的软件分到一个组里，按个快捷键秒开；做设计的人可以把素材文件夹关联进来；程序员能把常用命令和代码库路径都收纳起来。这些都是它官方列出来、也比较贴合实际使用的场景。

## 怎么下载安装 Dawn Launcher

安装流程不复杂，按下面的步骤走就行：

1. **打开官网下载页**，选择适合自己系统的版本（目前支持 Windows 7 及以上，最新版本仅支持 Windows 10 以上）
2. **运行安装包**，按提示完成安装，或者选择便携版直接解压使用
3. **首次启动**，会看到一个朴素简洁的主界面，左侧是分类栏
4. **右键侧边栏**，新建分类或子分类，按自己的使用习惯分组
5. **拖拽添加项目**：把软件快捷方式、文件夹或网址拖进对应分类里
6. **设置呼出快捷键**：在设置里绑定一个不和其他软件冲突的组合键，比如 Ctrl+空格

> 便携版用户注意：数据默认存在软件根目录的 data 文件夹里，更新版本前记得手动备份一下，别把分类数据搞丢了。

装完第一次用，说实话界面朴素到有点意外——没有花里胡哨的引导动画，直接就是一个能用的工具。喜欢简洁的人会觉得清爽，期待"全能工具箱"体验的人可能会觉得功能偏少。这两种感受都正常，取决于你要解决的是什么问题。

## Dawn Launcher 1 和 Dawn Launcher 2 有什么区别

这是网上搜索量比较大的一个问题，这里说清楚。

Dawn Launcher 目前分两个版本线：**Dawn Launcher 1** 是早期版本，目前仍可免费使用，但官方基本只做 bug 修复，不再添加新功能。**Dawn Launcher 2** 是后续推出的版本，持续在更新，多出来的功能集中在自定义和高阶操作上。

| 功能项 | Dawn Launcher 1 | Dawn Launcher 2 |
|---|---|---|
| 一键启动软件 / 文件 / 网址 | ✓ | ✓ |
| 快速搜索 / 网络搜索 | ✓ | ✓ |
| 子分类多组管理 | ✓ | ✓ |
| 关联文件夹 | ✓ | ✓ |
| 分类自定义字体 / 图标 / 密码 | — | ✓ |
| 分类独立窗口展示 | — | ✓ |
| 设置环境变量 | — | ✓ |
| 数据自动备份 | — | ✓ |
| 后续功能更新 | — | ✓ |

简单总结一下：如果你只是想让桌面清爽一点，不在乎分类加密、自动备份这些细节功能，**Dawn Launcher 1 的免费版基本就够用**。如果你是重度用户，看重长期更新和进阶功能，才需要考虑 2 的付费版本。

**这里要提醒一点**：Dawn Launcher 2 涉及付费授权，目前市面上有不少第三方商城在转售所谓的"激活码"，价格、授权条款各不相同。我没有找到这些转售渠道与开发者官方合作关系的独立第三方证实，所以这篇文章不会替你推荐具体购买渠道——如果你决定要买 2 的付费版本，建议直接通过软件内的购买入口或开发者官网确认的渠道完成，避免买到来源不明的授权码。免费的 Dawn Launcher 1 不存在这个问题，可以直接放心下载使用。

## 和同类工具比，Dawn Launcher 值得选吗

市面上做桌面整理 / 快捷启动的工具不少，比较常见的有 Fences、uTools、Rolan、酷呆桌面这些。它们风格差异挺大。

Fences 走的是"栅栏"思路，把桌面图标框进可折叠的区域里，视觉效果更花；uTools 是工具箱型产品，插件生态丰富，功能边界比 Dawn Launcher 宽很多；Rolan 体量更轻巧，适合只要一个极简呼出面板的人。

异次元软件世界的一篇评测里提到，Dawn Launcher 整理桌面快捷方式、保持桌面整洁的思路简单直接，操作直观方便，能轻松添加程序快捷方式、盘符、文件夹、网站，整体简单易用，不算功能强大丰富，但风格朴素简洁，适合不需要太复杂功能的用户。这个评价我自己用下来也比较认同——它不是万能工具箱，就是把一件事做好：把乱的图标变整齐。

换句话说，**如果你要的是简单够用、上手没有学习成本的整理工具，Dawn Launcher 是个合适选择；如果你要的是带插件生态、能干很多杂事的全能工具箱，那 uTools 这类产品可能更贴近需求。**

## 常见问题

**Dawn Launcher 支持 macOS 吗？**
目前不支持，只能在 Windows 上运行。作者早期提到过跨平台的计划，但截至目前没有看到 Mac 或 Linux 版本正式发布。

**Dawn Launcher 是免费软件吗？**
Dawn Launcher 1 完全免费，且在 GitHub 上开源，可以查看源码。Dawn Launcher 2 是后续版本，包含付费授权的高级功能。

**换电脑或重装系统后，数据还能用吗？**
便携版的数据存在软件根目录的 data 文件夹里，换电脑时把这个文件夹一起拷过去就行。装机版建议提前手动备份分类数据。

**呼出快捷键可以自己设置吗？**
可以，在设置里自由绑定组合键，比较常见的设置是 Ctrl+空格或 Alt+空格，选一个不和其他常用软件冲突的就行。

**Dawn Launcher 和音速启动是什么关系？**
没有官方关联。作者公开说过，Dawn Launcher 是参考多年前"音速启动"这款工具的思路，自己重新开发的一款独立软件。
